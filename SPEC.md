# SPEC.md - Диалайн Интернет Провайдер

## 1. Project Overview

- **Project name**: Диалайн - Интернет провайдер
- **Type**: Corporate landing page with interactive features
- **Core functionality**: Современный сайт с калькулятором подбора тарифа и формой онлайн-подключения
- **Target users**: Физические лица, ищущие интернет-услуги

## 2. UI/UX Specification

### Layout Structure

- **Header**: Логотип + навигация + контактный телефон
- **Hero**: Главный баннер с слоганом и CTA кнопкой
- **Tariff Calculator**: Интерактивный калькулятор выбора тарифа
- **Tariffs**: Карточки тарифов
- **Connection Form**: Форма онлайн-подключения
- **Footer**: Контакты, соцсети, копирайт

### Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Visual Design

**Color Palette**:
- Primary: `#0A2647` (глубокий синий)
- Secondary: `#144272` (средний синий)
- Accent: `#2C74B3` (яркий голубой)
- Highlight: `#205295` (акцентный)
- Light: `#F0F6FA` (светлый фон)
- White: `#FFFFFF`
- Text: `#1A1A2E` (тёмный текст)
- Text Light: `#6B7280`

**Typography**:
- Headings: 'Unbounded', cursive - современный геометрический гротеск
- Body: 'Onest', sans-serif - читаемый современный шрифт
- H1: 56px / bold
- H2: 40px / bold
- H3: 24px / semibold
- Body: 16px / regular

**Spacing**:
- Section padding: 100px vertical
- Container max-width: 1200px
- Card padding: 32px
- Gap: 24px

**Visual Effects**:
- Box shadows: `0 20px 60px rgba(10, 38, 71, 0.1)`
- Border radius: 20px для карточек, 12px для кнопок
- Hover transitions: 0.3s ease
- Gradient overlays на hero

### Components

**Header**:
- Фиксированный, с blur фоном при скролле
- Логотип (текстовый)
- Навигация: Тарифы, Калькулятор, Подключение
- Телефон с иконкой

**Hero Section**:
- Градиентный фон (синий переход)
- Анимированные частицы/волны
- Заголовок + подзаголовок
- Две кнопки: "Выбрать тариф" и "Подключиться"

**Tariff Calculator**:
- Слайдер выбора скорости (10-1000 Мбит/с)
- Переключатель типа подключения (Ethernet/GPON)
- Чекбоксы доп. услуг (IP-телефония, антивирус)
- Результат: рекомендуемый тариф + цена

**Tariff Cards**:
- 4 тарифа: "Старт", "Оптимальный", "Премиум", "Бизнес"
- Для каждого: название, скорость, цена, список возможностей
- Акцентная карточка "Популярный"
- Кнопка "Выбрать"

**Connection Form**:
- Поля: Имя, Телефон, Адрес, Тариф (выбор из списка)
- Валидация полей
- Успешная отправка с анимацией

**Footer**:
- Три колонки: О компании, Услуги, Контакты
- Соцсети
- Копирайт

## 3. Functionality Specification

### Tariff Calculator
- Слайдер скорости: 10, 30, 50, 100, 200, 500, 1000 Мбит/с
- Тип подключения влияет на доступные тарифы
- Доп. услуги добавляют стоимость:
  - Статический IP: +100 руб/мес
  - Антивирус: +50 руб/мес
  - Родительский контроль: +30 руб/мес
- Автоматический подбор тарифа по скорости
- Анимация пересчёта цены

### Connection Form
- Все поля обязательны
- Телефон: маска ввода
- При успехе: показ модального окна "Спасибо!"
- Данные сохраняются в localStorage

### Animations
- Плавный scroll к секциям
- Появление карточек при скролле (fade-in)
- Hover эффекты на кнопках и карточках
- Пульсация на CTA кнопке

## 4. Acceptance Criteria

- [ ] Страница загружается без ошибок
- [ ] Все шрифты загружены корректно
- [ ] Калькулятор работает: меняется цена при выборе опций
- [ ] Форма валидируется, показывает ошибки
- [ ] Адаптивная вёрстка работает на всех размерах
- [ ] Анимации плавные и не лагают
